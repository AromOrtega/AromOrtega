## 👨‍💻 Sobre mí

Soy un **Software Developer** enfocado en crear soluciones eficientes, escalables y de alto impacto. Me apasiona la arquitectura de software limpia y la integración de sistemas complejos, especialmente en el mundo de la robótica de servicio.

- 🔭 **Enfoque actual:** Desarrollo de soluciones integrales combinando un backend sólido (**.NET Core**) con hardware robótico.
- 💼 **Rol actual:** Desarrollador de Software en **Glexco Robotics & Automation**.
- 🌱 **Aprendizaje continuo:** Arquitecturas distribuidas, Docker, patrones de diseño (CQRS) y microservicios.
- ⚡ **Filosofía:** Escribir código limpio hoy para evitar problemas mañana. Liderar con el ejemplo y aportar valor real en cada despliegue.
